Gaia supports vector, raster, `WMS`, and `WFS` workflows in a single editing environment.

Layer operations include:

- Edit attributes with validation
- Draw and update points, lines, and polygons
- Run queries and inspect data directly in the workspace
- Apply live styling updates while editing

Data import is powered by GDAL and browser-native tooling, with support for formats such as Shapefile, GeoJSON, GeoTIFF, KML, GPX, and other common geospatial sources.
