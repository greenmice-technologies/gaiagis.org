Gaia includes a print layout editor for creating and managing print layouts for your maps.

With print layout editor you can:

- Create and manage print layouts
- Add and remove pages
- Add and remove elements
- Add and remove guides
- Add and remove rulers
- Add and remove grids
- Add and remove layers
- Add and remove styles
- Add and remove legends
- Add and remove scale bars
