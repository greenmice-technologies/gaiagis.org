# gaiagis.org

Public website repository for `https://gaiagis.org/`.

This repository contains derived public content only. The canonical product brief, capability map, and project status are maintained in the GAIA application repository:

- canonical source: `gaia/docs/readme.md`

## Content Contract

- This repo owns public marketing pages, feature summaries, CTA flows, and site navigation.
- This repo does not define new product capabilities independently from the canonical source.
- English is the maintained editorial baseline. Other languages should be published only after explicit editorial review.

## Local Commands

Run the website locally:

```bash
hugo server
```

Build the static site:

```bash
hugo
```
